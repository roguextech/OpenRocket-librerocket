What is /db/?

/db/ refers to the folder in which all of the program's non-code and non-asset data is stored. This leaves the file theme.json, and the several material and motor files. This is also the place where any downloaded or user-created presets will be stored, unless the user preset path is changed in /cfg/settings.json.

Materials.json

This file stores all of the materials known to the program. It also stores the properties of those materials. In this file there are three sections:

  • struct, which stores structural materials
  • line, which stores materials used in shock cords
  • chute, which stores parachute materials

Struct

The entries in this section take the form:
